# Open Activity Bar

## Finding GooseCode in VS Code

The GooseCode extension adds an icon to your VS Code Activity Bar (the vertical bar on the left side of VS Code).

<br/>

## GooseCode Views

Click the GooseCode icon to access:

### Connection
- Shows the current connection status
- Start/stop the extension server
- Copy the connection password
- Toggle localhost-only mode

### Code Sources
- Lists available workspaces
- Enable/disable workspaces for GooseCode
- Refresh the workspace list

### Active Session
- Visible when connected and generating
- Manage your current generation session
- Regenerate, step through, save, or discard

<br/>

## Quick Actions

- **Broadcast**: Temporarily advertise the service for easy device discovery
- **Copy Password**: Copy the connection password to clipboard
