# Your First Command

## Creating a Snippet

The simplest way to start using GooseCode is by creating a snippet from your code.

<br/>

## How to Create a Snippet

1. Select some code in your editor
2. Right-click to open the context menu
3. Navigate to **GooseCode > Snippet**

<br/>

## What Happens

- The selected code is sent to GooseCode
- A visual snippet is created in the GooseCode app
- You can organize and annotate your snippets

<br/>

## Keyboard Shortcuts

- **Snippet**: `Alt+Ctrl+S` (Windows) or `Alt+Ctrl+Cmd+S` (Mac)
- **Snippet Minimized**: `Alt+Ctrl+M` (Windows) or `Alt+Ctrl+Cmd+M` (Mac)

<br/>
