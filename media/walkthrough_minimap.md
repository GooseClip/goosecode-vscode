# Your Coding Minimap

## Overview

The GooseCode minimap provides a bird's-eye view of your code snippets, enabling quick navigation and visualization of your codebase structure.

<br/>

## Activating Minimap Mode

- Press **M** within GooseCode, or
- Run the `GooseCode: Toggle Minimap` command in VS Code

<br/>

## Minimap Features

- **Click** on snippets to navigate to that code location
- **Zoom (Z)** - Preview snippets while zoomed out
- **Focus (F)** - Center the view on the cursor position

<br/>

## Customization

- Resize and reposition the minimap window
- Position is restored on exit
- Use **Overlay (O)** mode for full GooseCode functionality overlaid on your screen
