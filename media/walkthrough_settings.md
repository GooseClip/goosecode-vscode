# Settings

## Start Automatically

Start GooseCode Extension server automatically on open.
> default: `false`
> recommend: `true`

<br/>

## Port

The port GooseCode will connect to the IDE.
> default: `60053`
> 
<br/>

## Localhost Only

Allow only localhost devices to connect.<br/><br/>
Set `false` if you plan to connect from another device e.g. iPad.<br/>
When setting up your connection on GooseCode, be sure to use your machines dial-out IP rather than your loopback IP.
> default: `true`

<br/>

## Password

Password protection to prevent unwanted connections.
> default: `GENERATED`
